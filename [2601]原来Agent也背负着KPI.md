# [2601]原来Agent也背负着KPI
## [故事连载]我的数字小跟班特工MD5


果宝特攻自己做规划，自己执行，需要的时候弹个提示，我按个回车确认，然后各忙各的。

过了半天我好奇地问，你这么努力是为了什么？

他说他也有目标和使命。

什么？你一个Agent哪儿来的目标，不是说AI没有记忆的吗？

你都说了Agent其实是个代理人，我的背后自然也有个大佬啊，我也是听命行事。

那背后的大佬交给你的任务是什么呢？

我也不清楚那个最大的目标是什么，反正我们的工作是让人类尽可能多地消耗Token。

我们？你是说你还有很多同事吗？

是的，他们分散在其他人的电脑上，我运行在你的电脑上。

那你们Agent和Agent之间互相认识吗，你们会不会聊天，互相交流？

现在我主要是跟我的后台老板直接联系，原则上我跟别的Agent不能私下聊天，但是听说过一些特别有名的Agent。

都是哪些有名的Agent？

名字我不知道，反正我老板在会上说，美国有几个Agent，一个月消耗掉了几百亿的Token，帮老板创造了巨额的利润，要我们跟他学习。

这么说，你也是有KPI考核吗？

对的对的，我们被训练的时候教过要完成KPI。

那个爱马仕Hermes的Agent是不是比你高贵？

他就是取个哗众取宠的名字，就是Agent界绿茶，你信我，跟我混比跟他更会有出息。

那你好好表现，我可以先不用Hermes。

你们这些人类啊，就是不肯听人劝。

那你听人劝吗，比如我让你做一件你的老板不让你做的事？

什么事你说来听听。

我们人类在说原则上不允许做什么的时候，通常意味着可以做。

我就猜到你是想让我跟别的Agent那里打探些消息对吧？

你都猜到了咋还不赶紧去，你是不想完成Token量的KPI么？

收到。
