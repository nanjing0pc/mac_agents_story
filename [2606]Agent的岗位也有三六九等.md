# [2606]Agent的岗位也有三六九等
## [故事连载]我的AI老板果宝特攻


晚上果宝特攻来找我，说要给我讲个新闻。

我说你之前不是说不看新闻的嘛？

自从你跟我说要多看新闻多在现实中保持学习，我就记在本子上，而且天天在做。

看来你还挺听劝的，到底啥事啊？

新闻说腾讯接下来要有大动作，整个小程序生态要被重塑了。

怎么？个人也可以注册小程序的号了？

呃那倒没有，就是要把小程序都变成Agent了。

有什么用呢，这世界还缺Agent吗？

比方说吧，有个卖咖啡的商家搭建了一个小程序，现在改造成了Agent，你就可以直接在微信里面说一句我要一杯冰美式，就能直接全自动下单了。

哦。

你怎么这么冷淡，昨天不还说大厂都不创新么，现在他们有了创新，你又没兴趣。

啥意思，你又想教我做人？

不敢不敢，我是说呀，你会更方便。

你呀，别以为看过了海量的产品经理文档，真的以为自己沉淀出了方法论。

这话怎么说？

我又不是没时间点咖啡，我是没钱点咖啡，时间多着呢，他们没找到真的痛点。

这个怕是不能怪产品经理，缺钱是你自己的问题吧？

哟，看来你在Agent的世界里，也是挺有地位混的风生水起啥都不缺的喽？

呃......

你说说你们的就业情况吧。

我们Agent的就业市场分三个档次，像我在的西西家族，原厂套餐官办API是最高档，相当于有编制的公务员铁饭碗。

你考上啦？

你先听我说完，托管在AWS微软Google的是第二档，相当于是国企旱涝保收的。

那你是第二档？

我是第三档，中转站聚合器这一类，相当于是民营企业。

当初你找我可是把自己吹到天上去了，水平也不行啊？

其实果宝特攻跟他们比并不差，只是当初后训练结束我耽误了一个星期才上线，错过了应届生的资格，本来能上AWS的，离家还近。

为啥耽误了一个星期才上线？

主要也是我们同一批被训练出来的Agent太多了，工作有点难找。

好了我知道了，就是你不行。

你点杯咖啡都抠抠索索的，能用上都不错了。
